儿科讲座：中性粒细胞缺乏性发热的诊疗

定义

  • 中性粒细胞(ANC)<0.5*10^9/L;严重型：中性粒细胞(ANC)<0.1*10^9/L ；
  • 原因：重症感染（毒力强、免疫力差）、化疗后（血液科常见）；

    中性粒缺乏发热

  • 标准：腋温>=38°C 或 >=37.7 °C持续1h以上（常见）
  • Risks：CVC，消化道粘膜炎，既往90d暴露于广谱抗菌药；
  • Common positions：肺、上呼吸道、肛周
  • 常不能明确病原菌，以G-菌为主：大肠埃希菌、肺炎克雷伯杆菌、铜绿假单胞菌

Diagnosis

  1. 病史
  2. 3d复查血常规

危险评估

  • High：
    • ANC严重减少 or 时间长
    • 合并症：
      • 血流动力学不稳
      • 消化道症状
      • 新发NS症状
      • 血管内导管感染
      • 肝肾功能不全：肝酶>正常上限5倍，肌酐清除率<30mg/L
  • Low: 7d内可恢复

复杂感染risks

  1. 休克、血流动力学不稳定、低血压
  2. 局灶性感染（肺炎、肠炎、CVC）
  3. 住院
  4. 长期严重营养不良（需早期干预）
  5. 并发症（出血、脱水、多器官衰竭）

耐药菌危险因素

  1. ESBL，MRSA
  2. 院内感染
  3. 长期感染
  4. 导尿管
  5. 接触过超广谱抗生素

中性粒细胞缺乏的处理

  1. 预防为主（降阶梯治疗）
  2. 注意体检、有创操作的感染
  3. 良好卫生习惯
  4. 综合治疗
  5. 支持治疗
  6. 积极抗感染

初始经验性抗感染治疗

  1. 低危患者：口服或静脉使用经验性抗菌药物
  2. 在门诊随访的低危患者：密切观察体征
  3. （化疗后）不能口服药物的患者：住院治疗
  4. 出现新感染灶：经验用药
  5. 高危患者：立刻住院

抗菌药物的调整：

  1. 发热48h无好转
  2. 抗菌无效，考虑特殊病原体以及真菌

抗感染治疗疗程

  1. 肺炎：10～21d
  2. 腹部：感染证据完全消失
  3. 金葡菌感染：首次血培养转阴性后14d
  4. MRSA：至少14d
  5. 耐甲氧西林肠球菌：5～7d

预防用药：

  • 高危患者：推荐
  • 低危患者：不推荐

降阶梯治疗：

  1. 联合—>单一；广谱—>窄谱；静脉—>口服；
  2. 并非在一开始制定完整的降阶梯治疗方案，而是指一开始用强效的方案，控制病情，而后减量或直接停药；
