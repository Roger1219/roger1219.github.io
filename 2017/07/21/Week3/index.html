Week3 妇产科门诊&B超室

7月17日

  • 24～28week常规做OGTT
  • 稽留流产：导致D-2聚体升高、纤溶亢进，继发出血倾向；

7月18日

  • 32～34week，膈肌上抬，挤压心脏，引起胸闷；
  • 宫颈机能不全：无腹痛，阴道流血，14～18week，测量宫颈长度（CL）
  • 双胞胎：发育较快，25week剖腹产手术；
  • 血糖升高，导致胎肺晚成熟；
  • OGTT筛查高危人群：双胎、体重>80kg，>35y；
  • 产检：第一次：22～24week，第二次：28～32week；
  • 羊水减少：1.胎盘老化，分泌减少，尽早剖出（孕早期原因）；2.胎儿发育不好，尿液生成减少（孕晚期原因）；注意复查BUS；
  • 母亲第二次妊娠，抗A、抗B抗体>512：提示有溶血存在；
  • 妊娠期高血压：20week以后：妊娠引起；20week以前：提示存在基础疾病或者其它系统引起的高血压；
  • 24week常规筛查甲状腺功能；

    7月19日

  • 子宫内膜：一般月经前最后10mm左右，绝经后5mm左右；
  • 不孕：首先考虑输卵管不通，碘油造影确认输卵管通畅后，使用促排卵药物；
  • IUD：阴道不规则流血、子宫内膜感染为常见副作用；
  • 在患者可能怀孕期间用药：注意对胎儿影响；
  • 尖锐湿疣病理：细胞挖空样改变、上皮呈乳头样改变；
  • 奥硝唑、头孢地尼：注意胃肠道反应，常先服用一种，另一种在月经期服用；
  • 钠氏腺囊肿：一般不用处理；
  • 子宫腺肌症：内膜增厚、无排卵、月经期疼痛，给予地屈孕酮；
  • 孕前检查：血常规、尿常规、肝肾功能、传染病、心电图、甲状腺功能；
  • 绝经后不规则阴道流血+子宫内膜增厚：诊断性刮宫
  • PCOS：长期不控制，可能转变为子宫内膜癌；
  • 子宫内膜炎：典型妇检子宫压痛，包括肌层炎症+内膜炎症；
  • 药💊流术后，残留：给予地屈孕酮促进内膜转化分泌期，10天后停药，月经来潮可能带走残留物；
  • 药流术前检查：血常规、白带常规、免疫十项、肝肾功能，胎停时间长（稽留流产）查凝血功能；
  • 滴虫：白带稀薄呈泡沫样，需要全身用药、夫妻同治（奥硝唑）、PV粉溶于水坐浴。
  • 霉菌：苏打粉溶于水坐浴、可局部用药、两者均可用消呋太尔；

7月20日

  • 人工流产（induced abortion）：
    1. 手术流产（负压吸引）
      • 适应症：妊娠10周内需要终止妊娠情况；
      • 受术者膀胱截石位，消毒外阴、铺巾；
      • 探测宫腔，用子宫探针探测子宫屈向河深度，若为感到阻力怀疑子宫穿孔，立刻停止手术；
      • 扩张宫颈：宫颈扩张器扩张宫颈管，一般扩张至大于所用吸管半号或1号；
      • 吸管负压吸引：待吸管进入宫腔后再开启负压（一般400mmHg负压吸引，此压力会损伤宫颈管引起术后宫颈黏连）。手拇指捏在吸引器开口下方，转动手臂，保证空腔每个位置均被吸引，一般按顺时针方向吸引1～2周。然后抽出吸引器（先弯折管子，消除吸引器负压，再抽出），排空管子内容物，然后再次进入吸引（防止管子因胚胎组织堵住没有吸力导致吸引不彻底）。
      • 待宫腔吸引干净，吸引器刮擦呈粗糙感，未吸引干净时呈光滑。
      • 拿走吸引盒所有内容物用滤网过滤，一定找出绒毛，确认胚胎完全吸出。
    2. 药物流产
      • 适应症：妊娠7周内，宫内妊娠、多次人工流产。
      • 米非司酮200mg顿服，第三天给予米索前列醇0.6mg；
      • 副作用轻；

7月21日

  • BPD：双顶径；HC：头围；AC：腹围；FL：股骨长；
  • 羊水指数：>8：正常,58：偏少,<5：过少；
  • 陶氏窝：又称道格拉斯腔，解剖学上指女性盆腔内子宫与直肠之间的间隙，属于盆腔,是盆腔的最低处。一般有盆腔炎会出现陶氏腔积液。陶氏腔积液在又称为盆腔积液。
  • B超：1cm深羊水～=100ml羊水。
