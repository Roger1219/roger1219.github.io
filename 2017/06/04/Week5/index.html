实习Week5

2017-06-04

Week5 儿科一

  • 7月31日

    • 传染性单核粒细胞增多症：单核cell or 淋巴cell 增多；

  • 8月1日

    • 血尿诊断思路：
      1. 鉴别真假血尿：肾源性or尿路， 镜下or肉眼
      2. 伴随症状：蛋白尿、水肿
      3. 尿蛋白检查：前夜吃干饭、少喝水
      4. 肾性血尿，无相关性肾炎：孤立性血尿or不典型肾炎，随访即可，避免感冒
      5. 检查24h尿钙尿肌酐
      6. 尿常规：蛋白+-，检查24h尿蛋白，更准确
    • 钙磷代谢检查思路：
      1. 血生化（血Ca2+，PO32-），25-(OH)2-D3
      2. PTH
      3. 甲状腺B超
      4. 24h尿钙

  • 8月2日

    • 胡桃夹综合症：胡桃夹现象也称左肾静脉受压，是指左肾静脉回流入下腔静脉过程中在穿经由腹主动脉和肠系膜上动脉形成的夹角或腹主动脉与脊柱之间的间隙内受到挤压，常伴有左肾静脉血流速度的下降、受压处远端静脉的扩张。当胡桃夹现象引起血尿、蛋白尿和左腰腹痛等一系列临床症状时，称为胡桃夹综合征

  • 8月3日

    • 幼儿心功能不全：
      1. 低龄儿（<6m)考虑遗传代谢性疾病：心肌病、心内膜弹性纤维增生症
      2. 检查：心肌酶谱、肌钙蛋白、心电图（安静时）

  • 8月4日

    • 心肌炎临床诊断标准
      1. 心功能不全
      2. 心脏扩大（X-ray 或 超声心动图）
      3. 心电图改变（以R波为主的2个或以上主要导联）：
        1. ST-T改变持续4天；
        2. 早搏；
        3. 传导阻滞；
        4. 异常Q波；
    • 高热惊厥
      • 一般只抽搐一次，在体温上升期出现。
      • 与中枢感染、癫痫鉴别
    • 菌血症：
      • CRP、PCT（降钙素原）升高；
      • 血培养阳性（必备）；
